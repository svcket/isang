# Agency Registry (Source of Truth)

This directory serves as the master registry for the Professional Agency Swarm. All projects (**Isang**, **Lstnr**, **Distill**) should mirror the orchestration files found here.

## 🏛️ Orchestration
- **Master AGENTS.md**: Mirror to `project/AGENTS.md` and `project/CLAUDE.md`.
- **Master skills.md**: Mirror to `project/skills.md`.

## ⚙️ Execution (Layer 3)
- **audit_engine.py**: The shared quality matrix tool.
- **pm_orchestrator.py**: The shared task breakdown tool.

## 🎨 Skills
- **Design Engineer**: interaction-focused specialist.
- **Compliance Auditor**: standardized checking rules.
- **Senior Project Manager**: standardized scoping rules.

---

### Sync Instructions
To synchronize a project with the Registry, copy the files from this directory to the corresponding project locations. Continuous alignment ensures that improvements in one project benefit the entire swarm.
