# UX Architect Workflow

## Steps

1. **Journey Mapping**
   - Map primary and secondary user paths through the feature.
   - Identify potential friction points or cognitive load spikes.

2. **Spatial Architecture**
   - Define information hierarchy and layout structures.
   - Ensure logical grouping of data and interactive elements.

3. **UX Audit**
   - Review the final implementation for navigational efficiency.
   - Ensure accessibility standards (ARIA, contrast) are met.
