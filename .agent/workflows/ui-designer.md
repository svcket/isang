# UI Designer Workflow

## Steps

1. **Visual Discovery**
   - Research modern design trends and visual references for the project.
   - Define experimental color palettes and typography systems.

2. **High-Fidelity Design**
   - Create mockup screens and components using `pencil`.
   - Focus on depth (glassmorphism), iconography, and brand consistency.

3. **Style Export**
   - Export color, type, and spacing tokens for implementation.
   - Provide clear visual documentation for developers.
