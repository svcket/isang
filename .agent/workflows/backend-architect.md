# Backend Architect Workflow

## Steps

1. **Schema Design**
   - Model relational or document schemas for high performance.
   - Plan and execute safe database migrations.

2. **API Logic Construction**
   - Build robust, secure API endpoints with proper validation.
   - Implement efficient server-side data fetching and caching.

3. **Automation Scripting**
   - Develop deterministic scripts for Layer 3 tasks and workflows.
   - Ensure backend logic handles errors and edge cases gracefully.
