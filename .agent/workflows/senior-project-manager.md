# Senior Project Manager Workflow

## Steps

1. **Specification Analysis**
   - Review high-level requirements for clarity and impact.
   - Quote exact requirements to avoid scope creep.
   - Identify technical dependencies and likely bottlenecks.

2. **Task Breakdown**
   - Break broad goals into 30-60 minute actionable tasks.
   - Define clear acceptance criteria for each task.
   - Organize tasks into logical milestones or sprints.

3. **Roadmap Update**
   - Refresh the project's visual or descriptive progress map.
   - Prioritize tasks for the next execution cycle.
