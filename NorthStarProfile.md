# Isang NorthStar Profile

## Project
Isang — High-Fidelity AI Travel Companion & Itinerary Architect.

## Core Purpose
Build the world's most intuitive and aesthetically premium AI travel planner. Isang is not a generic chatbot; it is a specialized agent that transforms vague travel intent into high-fidelity, actionable, and culturally authentic journeys.

### Its job is to:
- **Interpret Intent**: Understand the nuance of budget, vibe, and cultural interest beyond simple keywords.
- **Architect Itineraries**: Create logical, geographically sensible, and exciting travel plans.
- **Provide Depth**: Offer regional authenticity (Regional Airline mockups, local currency context, "Cultural Intelligence").
- **Visual Delight**: Present data through a "WOW" UI that feels like a premium travel magazine.

## North Star Outcome
Every interaction should make the user feel:
> "This isn't just an AI; it's a travel expert that has actually been there. The interface is beautiful, and the plan is ready for me to book."

## Quality Standards (The "Senior Assistant" Bar)
- **Visual Excellence**: No browser defaults. Use curated HSL palettes, smooth transitions, and glassmorphism.
- **Structural Integrity**: Consistent card dimensions, sticky headers, and logical scroll continuity.
- **Data Grounding**: No fake destination info. Use real-world logic for travel times, regional airlines, and seasonal vibes.
- **Proactive Nudging**: Anticipate the next user question (e.g., "Add to Trip," "Check Visa," "See Hotels").

## Implementation Philosophy (G-Stack)
- **Ideation -> CEO Gear**: Question if the feature is ambitious enough.
- **Planning -> Eng Manager Gear**: Verify architectural boundaries before coding.
- **Execution -> DOE Layer 3**: Use deterministic execution for complex data/UI logic.
- **Verification -> QA Gear**: Paranoid testing of every visual and functional detail.

## Tone & Brand
- **Premium**: Sleek, modern, and reliable.
- **Authentic**: Culturally aware and detail-oriented.
- **Agentic**: Proactive and helpful without being robotic.
